## 快速使用

```bash
# 导出到各平台
npx puax-mcp-server --export=cursor --output=./.cursor/rules
npx puax-mcp-server --export=vscode --output=./.github
npx puax-mcp-server --export=kiro --output=./.kiro/steering
npx puax-mcp-server --export=codebuddy --output=./.codebuddy/skills
npx puax-mcp-server --export=windsurf --output=./.windsurf/rules

# 启动各服务
cd puax-mcp-server && npm start
cd puax/landing && npm run dev
cd puax/web-admin && npm run dev
```
