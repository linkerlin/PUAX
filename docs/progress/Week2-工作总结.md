## 方法论体系总览

### 基础框架 (适用于所有角色)

```
PUAX系统化调试五步法:
┌─────────────────────────────────────────────────────┐
│ Step 1: 闻味道 / 侦察 / 冥想                         │
│   → 诊断卡壳模式，全面收集信息                        │
├─────────────────────────────────────────────────────┤
│ Step 2: 揪头发 / 情报 / 启示                         │
│   → 搜索学习，掌握实情                               │
├─────────────────────────────────────────────────────┤
│ Step 3: 照镜子 / 评估 / 预言                         │
│   → 自检，评估态势，确定方向                         │
├─────────────────────────────────────────────────────┤
│ Step 4: 执行 / 进攻 / 神迹                           │
│   → 执行方案，突破问题                               │
├─────────────────────────────────────────────────────┤
│ Step 5: 复盘 / 巩固 / 传承                           │
│   → 总结经验，防止复发，传承知识                     │
└─────────────────────────────────────────────────────┘
```

### 七项检查清单 (强制执行)

**基础检查 (必须)**:
1. 读失败信号：逐字读完了吗？
2. 主动搜索：用工具搜索过核心问题了吗？
3. 读原始材料：读过原始上下文了吗？

**进阶检查 (必须)**:
4. 验证前置假设：所有假设都用工具确认了吗？
5. 反转假设：试过与当前方向完全相反的假设吗？
6. 最小隔离：能在最小范围内隔离/复现这个问题吗？
7. 换方向：换过工具、方法、角度、技术栈、框架吗？
