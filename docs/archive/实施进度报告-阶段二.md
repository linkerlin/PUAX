## 🎯 功能使用示例

### 1. 分级角色使用

```typescript
import { roleLevelManager } from 'puax-mcp-server/src/role-levels';

// 获取推荐分级
const recommendation = roleLevelManager.recommendLevel(
  'complex',      // 任务复杂度
  5,              // 团队规模
  'organization'  // 范围
);

console.log(recommendation.recommendedLevel);  // 'p10'
console.log(recommendation.suitableRoles);     // ['strategic-architect', ...]
```

### 2. Agent Team 协作

```typescript
import { agentTeamManager } from 'puax-mcp-server/src/role-levels/agent-team';

// 创建团队
const team = agentTeamManager.createTeamFromTemplate(
  'sprint-team',
  '微服务重构项目'
);

// 分配任务
const task = agentTeamManager.assignTask(team.id, {
  type: 'development',
  description: '实现用户服务',
  assignedTo: 'military-warrior',
  priority: 'high',
  dependencies: []
});

// 生成协作剧本
const script = agentTeamManager.generateCollaborationScript(team.id);
```

### 3. 方法论路由

```typescript
import { methodologyRouter } from 'puax-mcp-server/src/core/methodology-router';

// 获取推荐方法论
const result = methodologyRouter.route({
  taskType: 'debugging',
  failureMode: 'spinning',
  attemptCount: 3,
  previousMethodologies: ['alibaba-closed-loop']
});

console.log(result.selectedMethodology);  // 'huawei-rca'
console.log(result.executionSteps);       // 五步法步骤
```

### 4. 反馈收集

```typescript
import { feedbackCollector } from 'puax-mcp-server/src/core/feedback-system';

// 提交角色反馈
feedbackCollector.submitRoleFeedback({
  roleId: 'military-warrior',
  sessionId: 'session-123',
  ratings: {
    helpfulness: 5,
    relevance: 4,
    quality: 5,
    overall: 5
  },
  wouldUseAgain: true,
  context: {
    taskType: 'debugging',
    triggerType: 'consecutive_failures',
    triggerLevel: 3,
    attemptCount: 3
  }
});

// 获取统计报告
const report = feedbackCollector.generateReport(30);
console.log(report);
```
