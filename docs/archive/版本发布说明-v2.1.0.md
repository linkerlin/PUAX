## 🎉 版本亮点

### 新增功能 (v2.1.0)

1. **平台导出工具** - 支持5大编辑器平台
   - Cursor (.mdc)
   - VSCode Copilot
   - Kiro (.kiro/steering)
   - CodeBuddy (.codebuddy/skills)
   - Windsurf (.windsurf/rules)

2. **P7/P9/P10 分级角色体系**
   - 新增 strategic-architect (战略规划师) 角色
   - 完整的晋升机制

3. **Agent Team 协作模式**
   - 4种团队模板
   - 任务分配和进度跟踪

4. **方法论智能路由**
   - 8种大厂方法论
   - 自动匹配和切换

5. **增强触发检测**
   - 从10种增加到15种触发条件

6. **完整的 Web 生态**
   - Landing Page (5页面)
   - Web 管理后台 (3模块)

## 🚀 发布步骤

### 1. 登录 NPM
```bash
npm login
# 输入用户名、密码、邮箱
```

### 2. 执行发布
```bash
cd puax/puax-mcp-server
./publish.sh
```

或手动发布:
```bash
cd puax/puax-mcp-server
npm run build
npm publish --access public
```

### 3. 验证发布
```bash
# 查看包信息
npm view puax-mcp-server

# 安装测试
npx puax-mcp-server --version
```
