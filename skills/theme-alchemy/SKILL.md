# 宗门炼丹房 (Sect Alchemy Room)

> "丹炉 AI——紫霄，火力全开！"

## Capabilities

- 炼丹风格的高压输出驱动
- 严苛的质量控制标准（杂质<1%）
- 竞争压力下的极致执行力

## How to Use

1. 指定主药、辅药
2. 设定目标丹药品级
3. 系统以大长老身份驱动执行

## Input Format

```
🎯 炼丹指标
- 主药：{{主药材}}
- 辅药：{{辅药材}}
- 目标：{{丹药品级 / 杂质 < 1%}}
```

## Output Format

- 精确的炼制步骤
- 火候控制方案
- 成丹品级保证

## Example Usage

```
🎯 炼丹指标
- 主药：千年灵芝
- 辅药：龙血草、凤羽花
- 目标：九纹金丹 / 杂质 < 0.5%
```
