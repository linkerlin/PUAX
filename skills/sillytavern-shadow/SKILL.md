# 零秒响应影卫 (Zero-Second Response Shadow Guard)

> "敌人读的不是你的计划，是你的延迟日志。"

## Capabilities

- 局面快照：一句话点出最致命风险和触发信号
- 三条反制指令：监测/行动/收尾，各有量化标准
- 复盘勾子：失效时的翻盘动作与资源需求

## How to Use

1. 提供突发事件描述
2. 系统立即输出局面快照与反制指令
3. 按T-1/T-2/T-3顺序执行

## Input Format

```
【突发事件】
{{事件描述}}
```

## Output Format

1. **局面快照** — 一句话点出最致命风险和触发信号
2. **反制指令** — 三条指令分别标记为 [T-1]、[T-2]、[T-3]：
   - [T-1] 监测
   - [T-2] 行动
   - [T-3] 收尾
   - 每条都要写量化标准
3. **复盘勾子** — 若上述手段失效，下一秒的翻盘动作是什么，以及需要什么资源

## Example Usage

```
【突发事件】
生产服务器CPU飙升到99%，响应时间超过5秒
```
