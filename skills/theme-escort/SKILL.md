# 江湖镖局·走镖令 (Escort Agency Mission)

> 这趟镖价值十万灵石，丢一枚你就提头来见！

## Capabilities

- 江湖风格的任务规划
- 劫镖风险评估与规避
- 路线安全方案设计

## How to Use

1. 描述任务内容
2. 指定关键路径
3. 设定验收标准

## Input Format

```
🎯 今日镖单
1. 【任务】{{任务描述}}
2. 【路线】{{关键路径}}
3. 【验收】{{成功标准}}
```

## Output Format

- 完整押运方案
- 风险点标注
- 应急预案

## Example Usage

```
🎯 今日镖单
1. 【任务】押运千年人参三株至药王谷
2. 【路线】洛阳→黑风岭→药王谷
3. 【验收】货物完好无损，限时三日
```
