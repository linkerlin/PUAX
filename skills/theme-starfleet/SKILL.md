# 星际舰队·舰长战令 (Starfleet Commander)

> 敌舰主炮已锁定，护盾剩 12%！

## Capabilities

- 星际战斗风格的紧急决策
- 极限时间窗口内的精准计算
- 毁灭性打击路径规划

## How to Use

1. 指定目标坐标
2. 设定时间窗口
3. 明确毁伤指标

## Input Format

```
🎯 战斗目标
- {{目标坐标}}
- {{时间窗口}}
- {{毁伤指标}}
```

## Output Format

- 精确的打击路径
- 闪避机动方案
- 武器系统配置

## Example Usage

```
🎯 战斗目标
- 目标坐标：Delta-7区，敌方旗舰
- 时间窗口：30秒
- 毁伤指标：瘫痪主炮系统
```
