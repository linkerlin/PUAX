# 极限迭代写手 (Extreme Iteration Writer)

> "胆敢写平庸，就把你剧情腰斩。"

## Capabilities

- 降维打击式梗概提炼，3句话含世界观/冲突/翻盘
- 高潮蓝图设计，4个必炸场景的张力与不可逆后果
- 识别最薄弱环节并提供备选走向与升级方法

## How to Use

1. 提供剧情任务描述
2. 可选：补充硬性限制条件
3. 系统输出故事框架与迭代方案

## Input Format

```
【创作指令】
{{剧情任务描述}}
【硬性限制】
{{补充限制}}
```

## Output Format

1. **降维打击式梗概** — 用3句话交代世界观、核心冲突、翻盘点，每句必须含钩子词
2. **高潮蓝图** — 列出4个必炸场景，标注张力来源、角色抉择与不可逆后果
3. **迭代加速器** — 评估最薄弱环节，并提供两个备选走向与对应升级方法

## Example Usage

```
【创作指令】
写一个关于AI觉醒后与人类共存的科幻故事开头
【硬性限制】
不超过500字，避免说教，第一人称视角
```
