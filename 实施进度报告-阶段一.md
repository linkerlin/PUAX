## 🚀 功能使用示例

### 1. 使用 MCP Sampling 主动触发

```typescript
import { SamplingClient } from 'puax-mcp-server/src/mcp/sampling-client.js';

const client = new SamplingClient(mcpClient, { cooldownMs: 30000 });

// 检测到触发条件后
const result = await client.executeSampling({
  sessionId: 'session-123',
  triggerType: 'consecutive_failures',
  confidence: 0.85,
  context: { attemptCount: 3 },
  recommendedRole: {
    id: 'military-warrior',
    name: '战士',
    systemPrompt: '你是战士角色...'
  }
});

if (result.accepted) {
  // 使用 result.injectedPrompt 注入到对话
}
```
