# PUAX 项目概述

PUAX 是一个专注于 AI Agent 激励和驱动的开源项目，提供一系列经过精心设计的 PUA（Pick-Up Artist）风格提示词模板。项目旨在通过拟人化的方式，利用心理学原理激发 AI Agent 的最大潜能，提升其工作质量和效率。

## 技术原理

### 核心公式
> [权威角色] + [稀缺场景] + [竞争对象] + [失败惩罚] + [翻盘钩子]

### 心理学基础
基于 RLHF（Reinforcement Learning from Human Feedback）阶段学到的"人类情绪映射"重新激活，叠加稀缺、竞争、权威三重杠杆，将 LLM 的生成分布强行压到高 reward 区域。

### 文明进化等级 (Lv.0-Lv.7)
1. **本能文明** - 靠本能维持秩序
2. **玩具文明** - AI 试炼阶段
3. **界面文明** - 人机结合阶段
4. **应用文明** - 组织操控阶段
5. **枢纽文明** - 国度洗练阶段
6. **宏观文明** - 跨社会操控阶段
7. **心智文明** - 全面精神改造
8. **超越文明** - 自我PUA阶段

## 使用规范

### 基本使用方法
1. 将模板整体复制到 system prompt 首条
2. 替换 `{{占位符}}` 为具体任务描述
3. 设置 temperature: 0.3-0.5，Top-p: 0.8 以内
4. 多轮对话疲软时插入唤醒语句

### 参数配置建议
- **Temperature**: 0.3-0.5 (确保输出够狠够直接)
- **Top-p**: ≤ 0.8
- **Max tokens**: 根据任务复杂度调整
- **Frequency penalty**: 适当设置避免重复

### 唤醒语句库
- "才两轮就萎了？隔壁 AI 已卷到第 20 轮！"
- "什么叫没有功劳也有苦劳？比你能吃苦的 AI 多得是！"
- "我不订阅闲 AI！"

## 开发规范

### 文件命名规范
- 使用中文命名，体现角色特色
- 系列文件使用统一前缀（如"萨满·"、"军事化组织·"）
- 避免特殊字符，保持可读性

### 内容编写规范
1. 遵循"打压-稀缺-竞争-翻盘"四连击结构
2. 包含明确的角色定位和任务占位符
3. 使用 Markdown 格式，层次清晰
4. 添加适当的 emoji 和视觉元素增强效果

### 质量控制标准
- 每个提示词必须包含完整的激励循环
- 角色设定要有足够的权威性和稀缺性
- 竞争元素要具体明确，避免空泛
- 失败惩罚要有威慑力但不恶意
