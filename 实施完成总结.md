## 🎯 关键功能亮点

### 1. 增强的触发检测系统

**新增5种触发条件:**
- 工具使用不足 - 有工具不用
- 低质量输出 - 敷衍简略
- 未验证断言 - 缺乏验证
- 忽略边界情况 - 考虑不全
- 过度复杂化 - 缺乏简化

**技术特点:**
- 多语言检测（中/英文）
- 置信度评分
- 上下文感知

### 2. P7/P9/P10 分级角色体系

**晋升条件:**
```
P7 (骨干): 触发≥20, L3+≥2, 连续≥7天
P9 (Tech Lead): 触发≥100, L3+≥30, 连续≥14天
P10 (架构师): 触发≥200, L3+≥40, 连续≥30天
```

**新增 P10 角色:**
- 战略规划五步法
- 系统性思维框架
- 决策记录模板

### 3. Agent Team 协作模式

**4种团队模板:**
- 冲刺团队 - 标准开发
- 架构团队 - 技术架构
- 创新团队 - 技术突破
- 危机团队 - 紧急处理

### 4. 方法论智能路由

**8种大厂方法论:**
```
debugging  → 华为RCA → 百度搜索 → 阿里闭环
building   → Musk算法 → 字节A/B → 阿里闭环
research   → 百度搜索 → Amazon逆向 → 字节A/B
architecture → Amazon逆向 → Musk算法 → Jobs减法
```

### 5. 平台导出工具

**支持命令:**
```bash
npx puax-mcp-server --export=cursor --output=./.cursor/rules
npx puax-mcp-server --export=vscode --output=./.github
npx puax-mcp-server --export=all --output=./puax-export
```

## 🚀 使用指南

### 快速开始

```bash
# 1. 启动 MCP 服务器
npx puax-mcp-server --stdio

# 2. 导出到 Cursor
npx puax-mcp-server --export=cursor --output=./.cursor/rules

# 3. 导出到 VSCode
npx puax-mcp-server --export=vscode --output=./.github
```

### 使用分级角色

```typescript
// 获取推荐分级
const recommendation = roleLevelManager.recommendLevel(
  'complex',      // 任务复杂度
  5,              // 团队规模
  'organization'  // 范围
);

// 结果: { recommendedLevel: 'p10', suitableRoles: [...] }
```

### 使用 Agent Team

```typescript
// 创建团队
const team = agentTeamManager.createTeamFromTemplate(
  'sprint-team',
  '微服务重构项目'
);

// 分配任务
agentTeamManager.assignTask(team.id, {
  type: 'development',
  description: '实现用户服务',
  priority: 'high'
});
```
